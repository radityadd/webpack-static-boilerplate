Buttons

Types

By default there are four types of buttons, Main, secondary, disabled, and low emphasis button. These four buttons also adaptable to use with icons, consider the icon size to be 24x24px to be fit into button.

Modification

(1) User may add glow to add emphasis, note this is not hover style, and just applicable to main button. (2) User may add icon to emphasize button function.

Guidelines

Padding Colors (1) Two main button uses same text color, only differs in background color, (2) when hovering, main text background overlayed with #00 20% vertical gradient, (3) low emphasis button uses same color with secondary button upon hovering.
